# Bandit Wargame Walkthrough

This repository documents my learning journey through the Bandit wargame from OverTheWire.

## 🎯 Objective
The goal of this project is to build a strong foundation in:
- Linux fundamentals
- Command line usage
- Problem-solving mindset
- Basic cybersecurity concepts

## 🧠 What is Bandit?
Bandit is a beginner-friendly wargame designed to teach the basics needed for cybersecurity and penetration testing.

Each level requires finding a password to access the next one using Linux commands and logic.

## 📂 Structure
Each level contains:
- Objective
- Commands used
- Explanation
- Key takeaways

## ⚠️ Disclaimer
This repository is for educational purposes only.  
All challenges come from OverTheWire and are designed to be solved in a legal environment.
